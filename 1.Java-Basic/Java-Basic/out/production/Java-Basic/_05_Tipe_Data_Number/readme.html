Konversi Tipe Data

Widening vs Narrowing, Widening otomatis terkonversi, Narrowing harus dikonversi sendiri

  • Widening : byte->short->int->long->float->double
  • Shortening : doubLe->float->long->int->short->byte

Hati-hati ketika melakukan Narrowing, bisa terjadi number Overflow
